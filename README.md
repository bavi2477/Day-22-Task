# Vite + React Project

This project demonstrates the usage of Vite along with React to create a simple subscription plan display.

## Introduction

This project utilizes Vite as the build tool and React for the frontend framework. It showcases a subscription plan display with different tiers (Free, Plus, and Pro) and their respective features.

## Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd <project-directory>
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000` to view the application.

## Usage

The main functionality of this application is to display subscription plans with their respective features. Each plan (Free, Plus, and Pro) has different features, and the UI highlights them accordingly. Users can hover over the cards to see the subscription details and click on the "Subscription" button to subscribe to a plan.
